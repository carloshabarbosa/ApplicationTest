ATM Application Test

Develop in Microsoft Visual Studio 2015 - .NET 4.5.2

User guide

- Open the project
- If your nuget is disbled, please enable it
- Build the project
- Execute the projet

- By default we have one valid user (you can create anothers):
	- admin

- Log in the system
- Navigate though the options


- Log path: $CURRENT_LOCATION/bin/Debug/log/log_{date}.txt - When debug
- Log path: $CURRENT_LOCATION/bin/Release/log/log_{date}.txt - When release
